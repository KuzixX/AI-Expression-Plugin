ПЛАН ЗАПИСИ ДАННЫХ ДЛЯ ОБУЧЕНИЯ FACEDENOISER
================================================
50 клипов, ~22 минут, ~40000 кадров (30 fps)


ИЗОЛИРОВАННЫЕ МЫШЦЫ
--------------------

Брови (6 клипов x 20 сек)
  1. Левая бровь вверх-вниз-нейтраль, повторять
  2. Правая бровь вверх-вниз-нейтраль
  3. Обе брови вверх-вниз
  4. Нахмуривание — сводишь брови к центру
  5. Левая вверх + правая вниз, потом наоборот
  6. Внешние края бровей вверх-вниз

Глаза (4 клипа x 20 сек)
  7.  Моргание левым глазом
  8.  Моргание правым глазом
  9.  Оба глаза — медленное закрытие и открытие
  10. Прищур (не закрывая полностью)

Рот (8 клипов x 20 сек)
  11. Улыбка только левой стороной
  12. Улыбка только правой стороной
  13. Симметричная улыбка — от легкой до максимальной
  14. Губы вниз (грусть) — левая, правая, обе
  15. Губы трубочкой
  16. Верхняя губа вверх (оскал)
  17. Нижняя губа вниз
  18. Уголки рта в стороны (натянутая улыбка)

Челюсть (3 клипа x 20 сек)
  19. Открытие рта — от закрытого до максимума, медленно
  20. Сдвиг челюсти влево-вправо
  21. Открытие + сдвиг одновременно

Iris (2 клипа x 20 сек)
  22. Глаза влево-вправо-центр, повторять
  23. Глаза вверх-вниз, по диагоналям, круговое движение


КОМБИНАЦИИ (ЭМОЦИИ)
--------------------

Базовые эмоции (7 клипов x 30 сек)
  24. Радость — улыбка + прищур + брови слегка вверх
  25. Грусть — уголки рта вниз + брови внутренние вверх
  26. Злость — нахмуривание + сжатые губы + прищур
  27. Удивление — брови вверх + рот открыт + глаза широко
  28. Отвращение — нос сморщен + верхняя губа вверх + прищур
  29. Страх — брови вверх + рот приоткрыт + глаза широко
  30. Презрение — одна сторона рта вверх, асимметрия

Сложные комбинации (5 клипов x 30 сек)
  31. Скептицизм — одна бровь вверх + легкая ухмылка
  32. Подмигивание — один глаз закрыт + улыбка
  33. Напряжение — все сжато: брови, губы, прищур
  34. Смех — открытый рот + прищур + брови вверх, с движением
  35. Задумчивость — губы в сторону + взгляд вверх


ПЕРЕХОДЫ
--------------------

(6 клипов x 40 сек)
  36. Нейтраль -> радость -> нейтраль -> грусть -> нейтраль (плавно)
  37. Удивление -> злость -> удивление -> отвращение
  38. Радость -> смех -> радость -> нейтраль
  39. Нейтраль -> все эмоции по кругу медленно
  40. Быстрые переключения между случайными выражениями
  41. Микровыражения — едва заметные движения от нейтрали


ЕСТЕСТВЕННАЯ МИМИКА
--------------------

(6 клипов x 45 сек)
  42. Рассказываешь историю вслух
  43. Читаешь текст с экрана вслух
  44. Реагируешь на воображаемый разговор (слушаешь, киваешь)
  45. Считаешь в уме, думаешь, вспоминаешь
  46. Смотришь "смешное видео" (естественные реакции)
  47. Просто сидишь 45 сек — idle мимика, моргания, микродвижения


ЭКСТРЕМАЛЬНЫЕ ЗНАЧЕНИЯ
--------------------

(3 клипа x 15 сек)
  48. Все на максимум — рот открыт, брови вверх, глаза широко
  49. Все сжато — зажмурился, губы сжаты, брови нахмурены
  50. Максимальная асимметрия — одна половина напряжена, другая расслаблена
